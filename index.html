Create and Upload an SSH Key to Github

To generate an SSH key and upload it to GitHub, follow these steps: Open a terminal on your computer and run the following command: ssh-keygen -t ed25519 -b 512 When prompted, enter a file in which to save the key. You can press Enter to accept the default file location. Next, you will be asked to enter a passphrase for the key. This is an optional security measure that adds an extra layer of protection to your key.
